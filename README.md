# Memleketim

Bu çalışma, temel HTML ögelerini kavramak için geliştirilmiştir. İçinde, aşağıdaki ögeler örneklendirilmiştir.
- `html`
  - `head`
    - `meta`
    - `title`
  - `body`
    - `header`
    - `main`
    - `footer`
    - `h1`, `h2`, `h3`
    - `section`
    - `p`
      - `em`
      - `strong`
    - `a`
    - `img`
    - `ol`, `ul`
      - `li`
    - `figure`
      - `figcaption`
