intro
  • HTML is the abbreviation for Hyper Text Markup Language
  • It is used to construct a web page
example
<!DOCTYPE html>
<html>
    <head>
        <title>Tab Title</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>
elements
  • <!DOCTYPE html> declares the document type
  • <html>

    is the root element

  • <head> covers all the meta elements
  • <title> is the title that is written on the browser's tab
  • <body> covers all the visible elements
  • <h1> is the largest heading
browsers

The browsers takes the HTML documents, interprets and visualizes them.

The most common used unes are Chrome, Edge, Firefox and Safari.

history
  • 1989 - Birth of HTML

    Tim Berners-Lee introduced the first version of HTML (Hypertext Markup Language) as a means to share and structure information on the World Wide Web.

  • 1995 - HTML2 Specification

    The HTML2 specification was published as a formal standard, introducing features like tables and image embedding, expanding the capabilities of HTML.

  • 1997 - HTML4 Specification

    The HTML4 specification was released, bringing significant advancements such as support for frames, form validation, and improved multimedia handling.

  • 2000 - XHTML Introduction

    XHTML (Extensible Hypertext Markup Language) was introduced as a reformulation of HTML in XML syntax, promoting stricter adherence to standards and enabling interoperability with other XML-based technologies.

  • 2014 - HTML5 Standardization

    HTML5 became an official standard by the W3C, introducing numerous new features and APIs such as semantic elements, multimedia support, canvas for drawing, local storage, and enhanced form controls.
